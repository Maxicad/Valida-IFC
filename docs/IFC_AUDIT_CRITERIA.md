# Criterios IFC Iniciais

- Lista universal inicial: `UNIVERSAL_IFC_CRITERIA.md`.
- CSV importavel atual: `../samples/criteria/criterios_universais.csv`.

- `IFC-001`: versao minima IFC4 ou IFC4X3.
- `IFC-002`: existencia de IfcProject.
- `IFC-003`: existencia de IfcSite.
- `IFC-004`: ao menos um IfcBuildingStorey.
- `IFC-005`: IfcSpace com Name preenchido.
- `IFC-006`: IfcDoor com Tag preenchida.
- `IFC-007`: GlobalId unico.
- `IFC-008`: elementos principais com geometria.
