# Relatorios

O relatorio HTML deve conter:

- Projeto e cliente.
- Arquivo IFC e schema.
- Data e responsavel.
- Quantidade de criterios avaliados.
- Percentual geral.
- Resultado por criticidade.
- Lista de inconformidades.
- Elementos afetados com GlobalId.
- Mensagem, valor encontrado, valor esperado e sugestao de correcao.

## Saidas

- HTML e impressao pelo navegador no MVP.
- PDF, CSV, XLSX e BCF em fases futuras.
